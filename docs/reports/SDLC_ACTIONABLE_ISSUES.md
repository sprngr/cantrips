**Problem**  
No CI workflow currently enforces test and structure quality gates.

**Scope**
- Add `.github/workflows/skills-quality.yml`
- Run all meta-skill script unit tests
- Run structure validation across all skills
- Enforce eval presence policy for required skills

**Acceptance Criteria**
- [ ] Workflow exists and triggers on PR + push to main
- [ ] Fails if any unit test fails
- [ ] Fails if frontmatter/structure validation fails
- [ ] Fails if required skills lack `evals/evals.json`
- [ ] README/TESTING docs include CI parity command

**Suggested Implementation Notes**

- Add lightweight Python gate script for eval presence policy
