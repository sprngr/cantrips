# Architecture Decision Records (ADR)

This directory tracks durable decisions for skills/meta-skills SDLC, quality gates, schema contracts, and governance workflows.

## Naming Convention

- `0001-<short-kebab-title>.md`
- Increment number for each new ADR.

## Status Values

- `Proposed`
- `Accepted`
- `Superseded by ADR-XXXX`
- `Deprecated`
