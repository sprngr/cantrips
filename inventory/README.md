# Inventory

Machine-readable catalog of all skills and agents in the repo.

## catalog.json

Auto-generated by `scripts/catalog.sh`. Do not hand-edit.

### Schema

```json
{
  "generated": "ISO 8601 timestamp",
  "sources": [{"name", "url", "type": "local|github", "version"}],
  "skills": [{"name", "path", "source", "version", "has_scripts", "has_references", "complexity"}],
  "agents": [{"name", "path", "source", "git_sha"}]
}
```

### Rebuild

```bash
./scripts/catalog.sh
```

Walks `skills/`, `agents/`, `archive/`. Writes `inventory/catalog.json`.

### Source types

| Type | Path / URL pattern |
|------|-------------------|
| `local` | Absolute path to cantrips (or other local repo) |
| `github` | `https://github.com/<owner>/<repo>` |
