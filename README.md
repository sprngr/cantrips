# 🔮 Cantrips

Because I'm still not convinced this isn't magic.

A personal repository of agents, skills, and skill playbooks — original creations and curated finds from the web — managed via lightweight scripts wrapping the [`skills`](https://www.npmjs.com/package/skills) npm package.

If you just want skills, they're available to install via `npx skills add sprngr/cantrips ---full-depth`.
